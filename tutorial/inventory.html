Inventory

Arth has powerful inventory managment system.Using Arth One can managed their inventory systematically.

Inventory Item

In Inventory Item one can view and mange their stock.

1. Item Dashboard

In item dashboard one can view their item summary.It contains Item Name, Group, Product Id, Re-order Point,Tax, HSN Code.

Stock History

In this software you can view stock history items. To view the stock history click the stock history button in the view column of item dashboard where you can find Batch No, Number of stocks, salling price, purchase price and expiary date

Item Transaction History

To view item transaction history click transaction history button where it display the all the transaction history made against that item

How To Edit Item Deatils?

To edit Item details click "Update Item" button. As soon as the button is clicked a update panel will appear in fornt of item dashboard where you can update the item details.

How To Delete Item?

To delete item you just click the 'Delete Item' button and by conforming delete action you can delete items. But make sure that you don't have any item stock in your inventory because by after delet an item you delete all the stock you have in your inventory

2. Create Item

To create an item in inventory click the 'Create Item' button. An item creation panel will appear infornt of item dashboard where item details can be enter. By clicking 'SAVE' button you can save the item in your inventory. You can view the new item in your dashboard

Inventory Group

"Inventory Group" where you can store your inventory systematically i.e. you can specify the location of your stock by storing your inventory stock in a group.

1. Group Dashboard

Group dashboard display all the groups and subgroups of your inventory.By default group dashboard have two groups Store and Godown.

2. Cretae Group

It is very easy to create a group. Click on the "Create New" button and fill up the form to create group. To create a primary group you should select under as uncategorized and to create Secondary Group or sub-group select the main group under which the sub-group you want to create

3. Delete Group

To delete a group you just right-click on the group that you want to delete and click delete to delete the specific group
